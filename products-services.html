Product Overview

The trains are newer, so shouldn't the service be better? Upon researching, we found out that the real problem with the inadequate train service is the decades-old train track system. Thus, MetroLive was born. We created software that tackles issues with track signaling, *add more*. But, these changes cannot be made with the current signaling system, so we worked with hardware engineers to upgrade the current mechanical system thus allowing our software to run smoothly and effectively.
